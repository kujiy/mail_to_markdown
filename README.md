# mail_to_markdown
Simple form which convert mail to markdown with just copy and paste especialy for Japanese

## Before
```
■その１
・ああああああああああああああ
・いいいいいいいいいいいい

その２
・ううううううううううううううううううううう
　ううううううううううううううう
・えええええええええええええええ

●見出し
○見出し
◎見出し
□見出し
◆見出し
◇見出し
```

## After
```
# その１
- ああああああああああああああ
- いいいいいいいいいいいい

# その２
- ううううううううううううううううううううう
　ううううううううううううううう
- えええええええええええええええ

# 見出し
# 見出し
...
```
**With two spaces all of end of lines.**
